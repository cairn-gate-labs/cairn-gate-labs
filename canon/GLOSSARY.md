# Cairn Gate Labs — Glossary

Defined terms used across Cairn Gate documentation. When a term appears in a doc, it means what's written here.

---

**Arm** — An internal operating division of Cairn Gate Labs LLC (Advisory, Systems, Tools, Briefs, Studio, IP). Arms are accounting categories, not legal entities.

**Brief** — A Cairn Gate published document: field notes, playbooks, research memos. Published under the brand. Never enables undecided IP.

**Charter** — The foundational document defining what Cairn Gate is, isn't, and is building toward. Everything else must be consistent with it.

**Clean-room policy** — The personal rules governing separation between Cairn Gate work and any employer obligations: personal devices, accounts, time, code, data only.

**Cortical memory architecture** — Cairn Gate's in-development AI memory system modeled on multi-scale human memory consolidation (working memory, episodic, semantic, procedural). Under active IP review — do not publish enabling details.

**Foundation docs** — Documents in `/foundation/`. Permanent identity. Changes require a numbered ADR. Annual review cadence.

**Invention log** — Date-stamped record of: what was conceived, what AI tools suggested, what was selected/rejected, what was implemented. Required before any IP filing or decision.

**Operating docs** — Documents in `/operating/`. Edited freely. Describe how the work happens right now.

**Provisional application** — USPTO provisional patent application: establishes filing date, permits "Patent Pending," no formal claims, 12-month window to file non-provisional.

**Strategy docs** — Documents in `/strategy/`. Living charter. Reviewed quarterly. Diffs expected.

**Thesis** — The falsifiable bet: the bottleneck in AI adoption is systems, not model capability. Specifically: organizations that build durable memory and workflow infrastructure will compound; those that don't stay in pilot mode.

**Workflow Review** — Cairn Gate's first Advisory offer: map client workflows, identify top 3 automation opportunities, deliver ROI/action plan.
